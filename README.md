# Plataforma Estudiantil FIEI

Plataforma web académica para la Facultad de Ingeniería Electrónica e Informática (FIEI).

## Tecnologías utilizadas
- **Frontend:** HTML5, CSS3, JavaScript (vanilla)
- **Backend:** PHP
- **Base de datos:** MySQL
- **Servidor local:** XAMPP (Apache)

## Funcionalidades
- Registro e inicio de sesión de estudiantes
- Panel de estudiante personalizado
- Biblioteca de recursos académicos con filtros y búsqueda
- Subida de archivos con drag & drop
- Formulario de contacto con almacenamiento en BD
- Panel de administración (gestión de usuarios, recursos y mensajes)

## Instalación
1. Clonar el repositorio en `htdocs` de XAMPP.
2. Importar `database.sql` en phpMyAdmin.
3. Configurar la conexión en `backend/config.php`.
4. Acceder desde `http://localhost/fiei-platform/`.
